The New Consultic.us

The site is now written locally in iA Writer, outputting markdown, cranked out to a static site with Jekyll and synchronized with Git hosted on GitHub Pages.

Badda-boom, Badda-bing!

What This Means

I'm current. I'm cool. I'm not (as) slow as I was when using PHP-based CMS workflows. I no longer need a database to run for a handful of static pages; the equivalent of a gas-guzzling SUV to drive to the mailbox.

What This Doesn't Mean

The new creation and deployment workflow doesn't change the need for proper design. I still like to start in Photoshop (sometimes Illustrator) to create a visual language.
